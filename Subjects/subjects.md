# Subjects

Each line defines a subject hashtag. You can nest tags with slashes.
Format: #tag - Description

## Core Subjects
#core/math - Mathematics
#core/reading - Reading
#core/writing - Writing
#core/history - History
#core/science - Science

## Electives
#elective/piano - Piano lessons
#elective/art - Art
#elective/pe - Physical Education
#elective/music - Music

## Other
#readaloud - Read-aloud time
#latin - Latin
#spanish - Spanish
#cooking - Cooking
#chores - Chores/Life skills
