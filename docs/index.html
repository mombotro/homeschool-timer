â“ FAQ

Can I use this with Obsidian?

Yes! On desktop, select your Obsidian vault as the project folder. All log files are plain markdown and will appear in Obsidian. You can edit them in either the timer app or Obsidian.

What if I lose my phone?

If you export JSON regularly, you can import it on a new device. Browser storage is persistent but backing up with JSON export is recommended.

Can multiple people use the same folder?

Yes, but not simultaneously. If sharing a folder (via Dropbox, etc.), coordinate who's using it to avoid conflicts. Better approach: each person exports JSON and one person imports/consolidates.

How do I delete a log entry?

Just edit the log in the editor - delete the line you don't want and click Save.

Can I edit hashtags after exporting CSV?

Yes! Open the CSV in Excel/Google Sheets and edit as needed. The source logs remain unchanged.
